# コマンドの判別
内部コマンド：bashに組み込まれているコマンド
外部コマンド：bashに組み込まれていないコマンド

typeコマンドで判別

setは内部コマンド
~~~
macaroni@PC130:~/projects/lpic_lv1$ type set
set is a shell builtin
~~~
envは外部コマンド(ファイルパスが表示されれば外部コマンド)
~~~
macaroni@PC130:~/projects/lpic_lv1$ type env
env is /usr/bin/env
~~~

# 外部コマンドの実行
外部コマンドを実行するとき、本来ならばファイルの絶対パスが必要。　
しかし、環境変数PATHにコマンドのパスが登録されている場合、絶対パスは不要。

これでも
~~~
/bin/env
~~~
これでも動く。
~~~
env
~~~

なぜなら環境変数PATHに/binが登録されているから。
~~~
echo $PATH
~~~

# 新しいコマンドパスを追加する
~~~
PATH=$PATH:newpath
~~~
