# cat
-nオプションをつけると
各行の左端に行番号

# nl
ファイル内容に行数を付けて表示
-bをつけると空の行にも数字が付く

# less
ファイルを１ページずつ表示

# head
ファイルの最初の10行分を表示

-nで表示する行数指定。

# tail
ファイルの最後の10行分を表示。
-nあり。
-f --followオプションでは、ファイルの末尾を表示した後、そのファイルに追記された内容を表示し続ける。

# cut
ファイルの中から情報を切り出して表示できる。
cut -d : -f 1
「:」文字で区切った、１番目を表示。

# sort
ファイルの中身を行単位で並び替えて表示。
デフォルトでは左側の文字を基準に昇順に

# uniq
ファイルの内容で重複した行を１行にまとめて表示。

例えば
apple
apple
banana
banana
banana
orange
ってファイルがあって

uniq ~.txt
apple
banana
orange
になる。

連続してるのしかまとめられない。

# od
octal dataの略。
デフォルトではファイルの内容を８進数で表示。
-tオプションと、--formatオプションで進数を指定可能。

-t xとすると16進数で表示

# wc
wor count
ファイルの内容を読み取り、その行数、単語数、文字数を表示。

wc /etc/passwd
  28   39 1430 /etc/passwd

行数、単語数、文字数。
左から行数、単語数、文字数。

-lで行数のみ
-wで単語数のみ
-cで文字数のみ

# tr
変換。
tr a b
aをbに変換

tr a b
abcd
bbcd

-dは指定した文字を削除

# split
ファイルを分割
